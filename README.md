# Т-Сберегатель

Т-Сберегатель это надежный цифровой ассистент, разработанный для защиты от импульсивных трат и помощи в принятии осознанных финансовых решений.

## Функциональности
*   Защита от импульсивных покупок
*   Категоризация расходов
*   Постановка финансовых целей
*   Уведомления о расходах

## Технологии
*   Python + Flask
*   SQLite + PostgreSQL
*   JavaScript
*   TypeScript + React
